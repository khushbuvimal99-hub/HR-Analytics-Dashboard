# 📊 HR Analytics Dashboard

## 🔍 Overview

This project analyzes employee data to identify attrition trends and workforce insights.

## 🎯 Objective

To help HR teams reduce attrition and improve employee performance.

## 🛠 Tools Used

* Microsoft Excel
* Data Visualization

## 📊 Key KPIs

* Total Employees
* Attrition Count
* Average Salary
* Years at Company
* Attrition Rate (%)
* Active Employees
* Avg Age
* Job Satisfaction Score

## 📈 Insights

* High attrition in specific departments
* Salary impacts employee retention
* Work-life balance is a key factor

## 📂 Project Files

* dashboard/hr_dashboard.xlsx
* data/raw_data.xlsx
* screenshots/

## 🚀 How to Use

Download Excel file and use slicers to explore insights.
